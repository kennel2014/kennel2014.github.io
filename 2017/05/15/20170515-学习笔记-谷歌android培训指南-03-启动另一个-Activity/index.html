putExtra() 方法将 EditText 的值添加到 Intent。Intent 能够以名为 extra 的键值对形式携带数据类型。键是一个公共常量 EXTRA_MESSAGE，因为下一个 Activity 将使用该键来检索文本值。为 Intent extra 定义键时最好使用应用的软件包名称作为前缀。这可以确保在应用与其他应用交互过程中这些键始终保持唯一。

    注：早期版本的 Android Studio 生成的 XML 布局可能不包括 android:id 属性。如果布局没有 android:id 属性，则调用 findViewById() 将失败。如果出现这种情况，打开 activity_display_message.xml 并将属性 android:id=”@+id/activity_display_message” 添加到布局元素。

    上面注中所说的一定要确认下自己的xml中有没有android:id属性，因为本人使用的android studio 2.3.1中没有自动生成android:id属性，自己添加的。
